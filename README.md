程序概述
  RPN（逆波兰表示法）计算器是一款基于栈结构的科学计算器，使用后缀表达式进行计算，无需括号即可处理复杂运算。该计算器采用C++实现，具有完整的错误处理和用户友好的交互界面。
功能特性
核心功能
  基本四则运算：加(+)、减(-)、乘(*)、除(/)
  高级数学函数：平方根(sqrt)、幂运算(^)
  特殊序列计算：斐波那契数列(fib)、杨辉三角求和(pascal)
  批量计算：一次性执行多个表达式
  历史记录：自动保存所有计算记录
用户体验
  双模式操作：演示模式 + 交互模式
  实时栈状态显示
  完整的错误提示
  历史记录管理
常见错误类型
错误类型	错误信息	原因	解决方法
栈操作数不足	错误: 栈中操作数不足	运算符需要的操作数不够	检查表达式格式
除零错误	错误: 除零错误	除数为0	修改表达式
负数平方根	错误: 不能对负数开平方	sqrt参数为负数	使用非负数
无效参数	错误: 斐波那契数列参数必须为非负整数	fib/pascal参数无效	使用非负整数
未知标记	错误: 未知的标记 'xxx'	输入了无效运算符	检查拼写
表达式不完整	错误: 表达式不完整或有多余操作数	表达式格式错误	检查表达式完整性
